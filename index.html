Clients

Quandos falamos em Clients, estamos falando do lado do consumidor, ou seja, o lado do usuário. Toda tecnologia envolvida deste lado visa focar na melhor interação para o usuário com o projeto desenvolvido. HTML, CSS e JavaScript, por exemplo, são ferramentas usadas para desenvolver o projeto destinado aos usuários. Com elas podemos criar sites estilizados e com interações dinânimcas focando na usabilidade de quem está navegando pelo seu site.

Servers

Ao se pensar em Servers, devemos imaginar o outro lado da moeda da explicação acima. Enquanto Clients é o lado do usuário, Server é a parte que lida com, por exemplo, hospedagem do site, é onde ficam armamzenados os arquivos (textos, audios, videos, etc...) daquele site que esta sendo acessado. É do lado Server tambem que se localizam os Databases, que são um tipo de servidor onde se armazenam dados, como por exemplo, dados dos cadastros de usuários de um site.

HTML

Afinal o que é o HTML? É uma linguagem de programação? HTML significa:

HyperText Markup Language, traduzido ao portugês, Linguagem de Marcação de Hipertexto.
Resumindo o HTML é um tipo de arquivo que os Navegadores conseguem ler e identificar o conteudo ali inserido, para reproduzir na tela do usuário. E NÃO, HTML não é uma linguagem de programação, uma vez que não pode criar interações dinâmicas. Apesar de não ter essa capacidade, o HTML pode fazer diversas funções interessantes através das TAGS que veremos a seguir.

Tags

Tags são propriedades/comandos existentes dentro do HTML que exercem funções especificas, dependendo da Tag utilizada. Tags são sempre usadas entre os sinais: maior/menor. Exemplo "< h1 >". Ao usar a tag "< h1 >" como no exemplo, criamos um título na nossa página web, onde todo escrito entre a Tag ficara em destaque (caixa de texto grande e em negrito, por se tratar de umm título). Vale lembrar tambem que as Tags geralmente são usadas no começo e no final do texto que se quer aplicar aquela propriedade. Exemplo: "< h1 > Insira um título aqui < /h1 >". Neste caso o texto escrito entre as Tags será o nosso título. Ao observarmos a tag utilizada, com uma barra "/" na frente, significa fechamento daquela Tag. Vale lembrar que nem todas as Tags necessitam fechamento, porem sua maioria sim.

HTML x CSS

O que são ? Bom, HTML foi explicado anteriormente. Agora podemos definir o CSS como a ferramenta que irá estilizar os arquivos HTML. "Comom aassim?". Vamos imaginar que meu arquivo HTML contenha textos e imagens, na minha página web esses conteúdos irão simplesmente aparecer na tela de forma crua. Ao utilizarmos CSS, podemos definir diversas características estéticas como: cor das letras do texto, fonte, tamanho, local da tela que ira aparecer, tamanho das imagens, bordas nas imagens, trasparência das imagens e muito mais. CSS tambem não é considerado linguagaem de programação!

Linguagens de programação

Agora sim chegamos neste tópico. Mas afinal o que é uma linguagem de programação? De acordo com o site "rockcontent.com" temos a seguinte definição:

É uma linguagem formal que, através de uma série de instruções, permite que um programador escreva um conjunto de ordens, ações consecutivas, dados e algoritmos para criar programas que controlam o comportamento físico e lógico de uma máquina.
Ou seja, são linhas de códigos escritas pelo programador para determinar uma série de funcionalidades em seu projeto. Cada linguagemm de programação têm sua forma de escrever os comandos e linhas de códigos, cada um com suas peculiaridades, porem pode se dizer que seguem uma mesma premissa, principalmente a lógica de programação envolvida.

São exemplos de linguagens de programação: JavaScript, Java, Ruby, Python e muitas outras.
